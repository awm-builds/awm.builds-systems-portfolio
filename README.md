# Portfolio

A modular, multi-project solution built on .NET 8,showcasing clean architecture principles,
modern web development practices, and professional engineering workflows.

This portfolio architecture is composed of:

- ASP.NET Core Web API (`Portfolio.API`)
- Razor Pages + Blazor front-end (`Portfolio.Web`)
- Domain models, business rules, and core abstractions (`Portfolio.Core`)
- Infrastructure layer for data access and external integrations (`Portfolio.Infrastructure`)

## Features

- Clean architecture with clear separation of concerns
- Extensible, testable, and maintainable project structure
- CI/CD-ready foundation suitable for GitHub Actions
- Conventional Commits and professional branching standards
